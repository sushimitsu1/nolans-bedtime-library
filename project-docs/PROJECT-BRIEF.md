# Project Brief

## Product
**Nolan's Big Vehicle Bedtime Library** is a mobile-friendly, installable static web app for parent-led bedtime reading.

## Audience
- Primary child: Nolan
- Target age: 3–5
- Parent reads aloud
- Goal: short, calming, vehicle-themed bedtime stories

## Core app behavior
- Search by title, vehicle, and keyword
- Browse by category:
  - Emergency Vehicles
  - Construction Vehicles
  - Monster Trucks
- Open one story and read page by page
- Large Previous / Next controls
- Page counter
- Favorites
- Shuffle / “Surprise Me Tonight”
- Parent settings to reset progress and preferences
- Optional bedtime mode and timer

## Deliberately excluded for now
- Read-aloud narration
- Automatic page advancement

Reason: the user wants to reduce screen time and keep the app parent-led.

## Hosting
- Static GitHub Pages site
