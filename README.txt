NOLAN'S BIG VEHICLE BEDTIME LIBRARY — MVP

QUICK START (Windows):
1. Extract the ZIP file.
2. Double-click START-WINDOWS.bat.
3. Open the address shown, usually http://localhost:8000

Alternative:
- Open index.html directly. Most features work, but install/offline mode requires the local server.

Controls:
- Tap/click page edges or Previous/Next.
- Swipe left/right on a phone or tablet.
- Use keyboard arrow keys.
- Reading progress is saved automatically in the browser.

Publishing later:
- Upload the full folder to GitHub Pages, Firebase Hosting, Netlify, or another static web host.
